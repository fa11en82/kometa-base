# kometa-base

Welcome to my shared Kometa script.

This is a basic (somewhat redacted) copy of my config for kometa daily runs.

In addition to this script I also keep an asset folder that has 4 sub-directories:
- Anime
- Collections
- Movies
- TV Shows

By keeping them organized this way I can add or replace posters, backgrounds, and title cards as necessary without having to dig too deeply.
